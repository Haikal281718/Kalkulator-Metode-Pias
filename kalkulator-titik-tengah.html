Kaidah Titik Tengah

Pendekatan integral tentu menggunakan nilai fungsi di titik tengah setiap subinterval. Lebih akurat dari Kaidah Segiempat untuk jumlah pias yang sama (orde O(h²)). Input h atau n saling otomatis menghitung satu sama lain.

Parameter

mᵢ = a + (i + 0.5) · h

Contoh cepat

⇌

Ubah n → h otomatis dihitung, ubah h → n otomatis dihitung.

Rumus aktif

I ≈ h · Σ f(mᵢ), i = 0, 1, 2, ..., n−1
Titik evaluasi mᵢ = a + (i + 0.5)·h — nilai tengah antara xᵢ dan xᵢ₊₁. Lebih akurat karena galat berpasangan saling mengurangi.
